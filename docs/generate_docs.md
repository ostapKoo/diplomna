Для проєкту "Tom and Jerry" використовується генератор `pdoc`, який створює документацію на основі Docstrings (стандарт Google Style) та анотацій типів (Type Hints).

## Як згенерувати документацію локально:
1. Переконайтеся, що `pdoc` встановлено: `pip install pdoc`
2. Виконайте команду в корені проєкту:
   `pdoc ./ --output-dir docs/html`
3. Згенерований веб-сайт з'явиться у папці `docs/html`. Відкрийте файл `index.html` у будь-якому браузері.
