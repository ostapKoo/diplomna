# Процедура оновлення (Update & Rollback)

## 1. Підготовка до оновлення
1. **Планування простою:** Оновлення вимагає зупинки Application Server та API. Тривалість простою: ~2 хвилини.
2. **Створення резервних копій:** Зробіть копію файлу `.env` та папки з нотатками користувача (якщо такі є).

## 2. Процес оновлення
1. **Зупинка служб:** Зупиніть процес `python main.py` та `uvicorn api:app` (через Task Manager або скрипт зупинки).
2. **Розгортання нового коду:**
   ```bash
   git fetch origin
   git pull origin main
   
# Оновлення конфігурацій та залежностей:

# Процедура відкату (Rollback)
1. Зупинити служби
2. скасувати зміни в git (git reset --hard HEAD~1)
3. Відновіть старі залежності: pip install -r requirements.txt
4. Запустіть служби та перевірте логи на відсутність помилок
